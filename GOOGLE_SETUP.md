1. Create a new Google Sheet.
2. Copy the spreadsheet ID from the URL.
3. Open Google Apps Script and create a new project.
4. Paste the code from google-apps-script.js into the script editor.
5. Replace YOUR_SPREADSHEET_ID with your real spreadsheet ID.
6. Deploy the script as a web app:
   - Execute as: Me
   - Who has access: Anyone
7. Copy the web app URL and replace YOUR_GOOGLE_APPS_SCRIPT_URL in script.js.
8. Reload the page and click Confirm.
